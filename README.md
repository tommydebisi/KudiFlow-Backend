# KudiFlow-Backend

This is the backend application designed for the **Kudiflow** application aimed at helping
students manage their income and expenses. Thereby, making students save more.

## Objectives
At the end of the project it should:
- Have a scalable backend server
- Authorize users using `JsonWebToken (JWT)`
- Have an API documentation to tell how the backend enpoints work.
- Implemnt google authentication to sign in and sign up users.

## Documntation
The API documentation containing examples can be found in [POSTMAN](https://documenter.getpostman.com/view/26131668/2s93RTQBvN)
